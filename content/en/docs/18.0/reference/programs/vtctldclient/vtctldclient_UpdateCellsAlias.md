Updates the content of a CellsAlias with the provided parameters, creating the CellsAlias if it does not exist.

Updates the content of a CellsAlias with the provided parameters, creating the CellsAlias if it does not exist.

```
vtctldclient UpdateCellsAlias [--cells <cell1,cell2,...> [--cells <cell4> ...]] <alias>
```
